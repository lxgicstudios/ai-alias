# ai-alias

[![npm version](https://img.shields.io/npm/v/ai-alias.svg)](https://www.npmjs.com/package/ai-alias)
[![npm downloads](https://img.shields.io/npm/dm/ai-alias.svg)](https://www.npmjs.com/package/ai-alias)

Generate useful shell aliases from your command history

## Install

```bash
npm install -g ai-alias
```

## Usage

```bash
npx ai-alias
```

## Setup

```bash
export OPENAI_API_KEY=sk-...
```
